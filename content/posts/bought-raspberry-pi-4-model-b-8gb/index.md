半導体不足と世界中の転売ヤーに狙われていることで、どこにも売っていないラズパイですが、[Raspberry Pi 4 Model B](https://www.raspberrypi.com/products/raspberry-pi-4-model-b/) 8GB の2台目を購入できました。自宅にあるセカンダリサーバーが [Raspberry Pi 3 Model B+](https://www.raspberrypi.com/products/raspberry-pi-3-model-b-plus/) で、通常運用時の性能は十分なのですが、Ansibleを実行するとプライマリサーバーの Raspberry Pi 4 Model B に比べてだいぶ遅く、Ansibleの実行時間の足を引っ張っていました。そのため、セカンダリサーバーも Raspberry Pi 4 にしたかったところでした。



Raspberry Pi 正規代理店の [Switch Science](https://www.switch-science.com/) で購入しましたが、入荷通知を受け取らないと購入は難しいです。私の場合、18:31に通知を受け取り、18:33に注文完了できたものの、18:45には売り切れていることを確認しました。私の経験では、18:30頃に通知が来ることが多いです。



iPhoneがあれば入荷通知が届いたタイミングで簡単にアラートを鳴らすことができます。



1. [商品ページ](https://www.switch-science.com/products/6370)で入荷通知登録をする
2. sales@switch-science.com をiPhoneの[連絡先](https://apps.apple.com/jp/app/id1069512615)アプリに追加
3. [メール](https://apps.apple.com/jp/app/id1108187098)アプリの右上の「編集」からVIPメールボックスを追加
4. VIPメールボックスの右側のⓘアイコンを選択
5. 「VIPを追加」から sales@switch-science.com を追加
6. 設定アプリの 通知 > メール > 通知をカスタマイズ > VIP を選択
7. 一番派手なサウンドを選ぶ



ケースにはMiuzeiを使っています。電源は良いものを使いたかっため、[Switch Science ラズパイ4に最適なACアダプター 5.1V/3.0A USB Type-Cコネクタ出力 SSCI-056830](https://www.switch-science.com/products/5683#erid10754076) を使用しています。
